<Warning>
Commands and binds are only available on the **Trickshot server**, not on vanilla servers (except `/die` and `/spectate`).
</Warning>

## Binds

| Action | Controller | Keyboard |
|--------|-----------|----------|
| Save Position | Up D-pad + Melee | N + Melee |
| Load Position | Down D-pad + Melee | 7 + Melee |
| Give Streaks | Left D-pad + Melee | 5 + Melee |
| Empty Mag | Left D-pad + Crouch | 5 + Crouch |

## Chat Commands

| Command | Description |
|---------|-------------|
| `/help` | Prints a list of chat commands |
| `/binds` | Prints binds |
| `/die` | Kills you (alias: `/sui`) |
| `/save` | Save your current position |
| `/load` | Load your saved position |
| `/reset` | Reset your saved position |
| `/drop` | Drop your current weapon |
| `/cs` | Drop canswap on the ground |
| `/overkill <weapon> <attachment>` | Gives any gun as a secondary (see [Overkill & Alt Weapons](/servers/hmw/overkill-alt)). Use `/overkill reset` to remove |
| `/alt <weapon> <attachment>` | Gives any gun as a third weapon (see [Overkill & Alt Weapons](/servers/hmw/overkill-alt)). Use `/alt reset` to remove |
| `/nv` | Toggle night vision from action slot |
| `/ammo` | Refills ammo and equipment |
| `/regen` | Toggle automatic ammo/equipment regeneration |
| `/empty` | Empties magazine leaving one bullet |
| `/streaks` | Gives equipped killstreaks. Can also give a specific streak (e.g., `/streaks <name>`) |
| `/dist` | Prints distance to crosshair in meters |
| `/spectate` | Puts you in spectator mode |
| `/discord` | Prints Discord server link |
| `!report <player> <reason>` | Report a player |

## VIP Commands

<Info>
VIP commands are available to players who have purchased CB Server VIP through the [Discord server](https://cbservers.xyz/discord).
</Info>

| Command | Description |
|---------|-------------|
| `/last` | Fast last |
| `/floaters` | Toggle floaters |
| `/random` | Gives you a random trickshotting class every time you spawn |
| `/rocket` | Toggle rocket ride |
| `/instashoot` | Toggle instashoot for trickshot weapons |
| `/ah` | Set your [afterhit weapon](/servers/hmw/afterhit) |
| `/votekick <player>` | Start a vote kick on a player |
