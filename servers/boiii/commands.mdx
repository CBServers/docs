## Binds

| Action | Controller | Keyboard |
|--------|-----------|----------|
| Save Position | Up D-pad + Melee | 1 + Melee |
| Load Position | Down D-pad + Melee | 2 + Melee |
| Give Streaks | Left D-pad + Melee | 5 + Melee |
| Give Specialist | Right D-pad + Melee | 4 + Melee |
| Drop Canswap | Left D-pad + Crouch | 5 + Crouch |
| Empty Mag | Down D-pad + Crouch | 2 + Crouch |

## Chat Commands

| Command | Description |
|---------|-------------|
| `/help` | Prints a list of chat commands and binds |
| `/die` | Kills you (alias: `/sui`) |
| `/save` | Save your current position |
| `/load` | Load your saved position |
| `/reset` | Reset your saved position |
| `/drop` | Drop your current weapon |
| `/cs` | Drop canswap on the ground |
| `/cg` | Gives glitched/special weapons (see [Afterhit Weapons](/servers/boiii/afterhit) for the list) |
| `/lo` | Saves your current weapon for respawn (use with `/cg`) |
| `/ammo` | Refills ammo |
| `/empty` | Sets current mag to 1 bullet |
| `/regen` | Toggle automatic ammo/equipment regeneration |
| `/streaks` | Gives scorestreaks |
| `/special` | Gives specialist weapon |
| `/boost` | Toggle unlimited boost jump |
| `/fov` | Toggle console FOV |
| `/dist` | Prints distance to crosshair in meters |
| `/spectate` | Puts you in spectator mode |
| `/discord` | Prints Discord server link |
| `/twitter` | Prints Twitter link |
| `!report <player> <reason>` | Report a player |

## VIP Commands

<Info>
VIP commands are available to players who have purchased CB Server VIP through the [Discord server](https://cbservers.xyz/discord).
</Info>

| Command | Description |
|---------|-------------|
| `/last` | Fast last |
| `/ufo` | Toggle UFO |
| `/ah <weapon>` | Set your [Afterhit weapon](/servers/boiii/afterhit) |
| `/floaters` | Toggle floaters |
| `/rocket` | Toggle rocket ride |
| `/bot` | Teleport bot to crosshair |
| `/bot reset` | Reset and unfreeze your frozen bots |
| `/votekick <player>` | Start a vote kick on a player |

<Warning>
Abusing the kick command to kick players who are not breaking rules will result in your VIP being revoked.
</Warning>
