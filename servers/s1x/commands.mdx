## Binds

| Action | Controller | Keyboard |
|--------|-----------|----------|
| Save Position | Up D-pad + Melee | N + Melee |
| Load Position | Down D-pad + Melee | M + Melee |
| Give Streaks | Left D-pad + Melee | 5 + Melee |
| Drop Canswap | Left D-pad + Crouch | 5 + Crouch |
| Empty Mag | Down D-pad + Crouch | 2 + Crouch |

## Chat Commands

| Command | Description |
|---------|-------------|
| `/help` | Prints a list of chat commands and binds |
| `/die` | Kills you (alias: `/sui`) |
| `/save` | Save your current position |
| `/load` | Load your saved position |
| `/reset` | Reset your saved position |
| `/drop` | Drop your current weapon |
| `/canswap` | Drop canswap on the ground (alias: `/cs`) |
| `/ammo` | Refills ammo and exo abilities |
| `/empty` | Sets current mag to 1 bullet |
| `/regen` | Toggle automatic ammo regeneration |
| `/streaks` | Gives scorestreaks |
| `/dist` | Prints distance to crosshair in meters |
| `/spectate` | Puts you in spectator mode |
| `/skip` | Vote to skip the current map |
| `/tk` | Gives throwing knife equipment (alias: `/throwingknife`) |
| `/tac` | Gives tactical insertion equipment (alias: `/tacticalinsertion`) |
| `/sa` | Gives steady aim perk (alias: `/steadyaim`) |
| `/grapple` | Gives grapple if on a supported map |
| `/cg <weapon>` | Gives a [custom gun](/servers/s1x/custom-guns) |
| `/lo save` or `/lo clear` | Saves or clears your primary and secondary for respawn with custom guns |
| `/discord` | Prints Discord server link |
| `!report <player> <reason>` | Report a player |

## VIP Commands

<Info>
VIP commands are available to players who have purchased CB Server VIP through the [Discord server](https://cbservers.xyz/discord).
</Info>

| Command | Description |
|---------|-------------|
| `/last` | Fast last |
| `/ufo` | Toggle UFO |
| `/ah <weapon>` | Set your [afterhit weapon](/servers/s1x/afterhit) |
| `/floaters` | Toggle floaters |
| `/rocket` | Toggle rocket ride |
| `/telbot` | Teleport bot to crosshair (alias: `/bot`) |
| `/votekick <player>` | Start a vote kick on a player |
