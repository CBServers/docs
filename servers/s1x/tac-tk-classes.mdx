Pre-built public and private match classes that come with Tactical Insertions and Throwing Knives already equipped.

## Download

[Download Premade Classes (ZIP)](https://github.com/CBServers/client-dowloads/raw/main/clients/other/S1x-Tacs-and-TK.zip)

## Installation

<Steps>
  <Step title="Extract the ZIP file">
    Extract the downloaded ZIP file.
  </Step>
  <Step title="Open your game folder">
    Go to your AW/S1x install folder.
  </Step>
  <Step title="Navigate to user folder">
    Open `players2` then the `user` folder inside it.
  </Step>
  <Step title="Copy the files">
    Place the two files from the ZIP (`mpdata` and `mpdata2`) into the `user` folder. Replace any existing files when prompted.
  </Step>
</Steps>
