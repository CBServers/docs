## Installation Guide

<Steps>
  <Step title="Download the torrent file">
    Click on the torrent link for your desired game above. This downloads a small `.torrent` file.
  </Step>
  <Step title="Install a torrent client">
    Download and install a torrent client such as [qBittorrent](https://www.qbittorrent.org/download) if you don't already have one.
  </Step>
  <Step title="Open the torrent">
    Open your torrent client, then go to **File > Add Torrent File...** and select the torrent file you downloaded.
  </Step>
  <Step title="Choose download location">
    Choose a location where you'd like to save the game files and start the download.
  </Step>
  <Step title="Wait for download">
    Wait for the download to complete. You'll know it's done when the status shows **Seeding** or **Completed**.

    <Note>
    This may take a while depending on your internet speed and the game size.
    </Note>
  </Step>
  <Step title="Download the game client">
    Download the game client for the game you downloaded. See our [game clients](/launcher#supported-game-clients) page.
  </Step>
</Steps>
