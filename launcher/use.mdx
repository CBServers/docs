## Managing Installs

The launcher can install or remove additional content for any game (Campaign, DLC, etc.). This works for both games you already have installed (from Steam or torrents) and games you installed through the launcher.

<Steps>
  <Step title="Navigate to the game">
    Click on the game in the sidebar.
  </Step>
  <Step title="Open manage install">
    Click the **Manage Install** button on the game page.
  </Step>
  <Step title="Select content">
    Check the content you want to install. Deselected components will be removed.
  </Step>
  <Step title="Apply changes">
    Confirm your selections and wait for the changes to complete.
  </Step>
</Steps>

## Verifying Game Files

If you're experiencing issues with a game, you can verify your game files:

<Steps>
  <Step title="Navigate to the game">
    Click on the game in the sidebar.
  </Step>
  <Step title="Click Verify">
    Click the **VERIFY** button on the game page.
  </Step>
  <Step title="Wait for verification">
    The launcher will check all files and redownload any corrupted or missing files.
  </Step>
</Steps>

## Support Page

The launcher has a support page at the bottom of the sidebar with quick links to:

- Our Discord Server
- Script to install required redistributables (Microsoft Visual C++ and DirectX)

<Warning>
Microsoft Visual C++ Redistributables and DirectX are **required** before playing any of the clients. Without these, you may experience crashes.
</Warning>

## Game Settings

Each game has a settings popup accessible by clicking the **Cog Wheel** on the game page. Here you can configure:

- Game install path
- Game launch behavior
- Game-specific options
- Launch arguments

## Launcher Settings

The launcher has its own settings page where you can configure:

- Launcher-specific settings
- Download server selection

### Download Server Selection

The default server selection is **Auto**, which chooses the fastest server. You can manually select a region from the dropdown:

- **Auto** (Recommended) - Automatically selects the fastest server
- **NA** - North America server
- **EU** - Europe server
