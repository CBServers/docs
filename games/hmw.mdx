## Prerequisites

- Modern Warfare Remastered from Steam or [Torrent](https://github.com/CBServers/client-dowloads/raw/main/torrents/h1_full_files.torrent)
- [cb-launcher.exe](https://github.com/CBServers/updater/raw/main/updater/cb-launcher/cb-launcher.exe) file

## Installation

<Steps>
  <Step title="Download the launcher">
    Download [cb-launcher.exe](https://github.com/CBServers/updater/raw/main/updater/cb-launcher/cb-launcher.exe).
  </Step>
  <Step title="Run the launcher">
    Run the launcher from any folder.
  </Step>
  <Step title="Select HorizonMW">
    Click on the **HorizonMW** tab and select **SETUP**.
  </Step>
  <Step title="Select game folder">
    Select your MWR Steam folder or torrented MWR folder.
  </Step>
  <Step title="Launch the client">
    Run the client by clicking **Run** and allow the game to update when prompted.
  </Step>
  <Step title="Play">
    The game should launch after the update completes.
  </Step>
</Steps>
