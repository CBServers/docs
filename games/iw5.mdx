<Note>
    Plutonium requires an account to play Multiplayer. You can create one at [forum.plutonium.pw/register](https://forum.plutonium.pw/register).
</Note>

<Tabs>
  <Tab title="Launcher Install">
    <Steps>
      <Step title="Download the launcher">
        Download [cb-launcher.exe](https://github.com/CBServers/updater/raw/main/updater/cb-launcher/cb-launcher.exe).
      </Step>
      <Step title="Run the launcher">
        Double-click `cb-launcher.exe` to start the launcher.
      </Step>
      <Step title="Select Modern Warfare 3">
        Click on the **Library** tab and find **Modern Warfare 3**.
      </Step>
      <Step title="Setup the game">
        Click the **SETUP** button and choose your install type:
        - **Existing install** - Select the folder where your Modern Warfare 3 files are located
        - **Download game** - Select a folder to download Modern Warfare 3 and choose content (Campaign, DLC, etc.)
      </Step>
      <Step title="Wait for installation">
        If downloading, wait for the game to finish. If you setup an existing install, click the **VERIFY** button to verify your files.
      </Step>
      <Step title="Play">
        Click **PLAY**, then select the mode you want to play and have fun!
      </Step>
    </Steps>
  </Tab>

  <Tab title="Manual Install">
    ## Prerequisites
    - Modern Warfare 3 from Steam (free to download via [steam://install/42750](steam://install/42750))
    <Steps>
      <Step title="Download the game">
        Download Modern Warfare 3 for free from [Steam](steam://install/42750).
      </Step>
      <Step title="Download the client">
        Download the appropriate client for your game mode:
        - **Multiplayer:** [plutonium.exe](https://cdn.plutonium.pw/updater/plutonium.exe)
        - **Singleplayer/Spec Ops:** [alterware-launcher.exe](https://github.com/alterware/alterware-launcher/releases/latest/download/alterware-launcher.exe)
      </Step>
      <Step title="Place in game folder">
        Place the downloaded exe in your Modern Warfare 3 game folder.
      </Step>
      <Step title="Launch the client">
        Open the client by double-clicking on the exe.
      </Step>
      <Step title="Wait for updates">
        All necessary files should download automatically.

        <Note>
        Plutonium may require you to set the install path in the launcher settings if the game is not detected automatically.
        </Note>
      </Step>
      <Step title="Play">
        Select your game mode and have fun!
      </Step>
    </Steps>
  </Tab>
</Tabs>
